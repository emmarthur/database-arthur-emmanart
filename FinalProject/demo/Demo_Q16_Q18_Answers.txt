-------------------------------------------------------------------------------
Q16 — INTERMEDIATE (join + GROUP BY + AVG — no CASE)
-------------------------------------------------------------------------------

Plain English question:
  Which countries had the highest average goals per match across all their leagues?

What the answer looks like:
  One row per country. Columns are country name and avg_goals_per_match (average of
  home_team_goal + away_team_goal over all matches in that country's leagues).

Keys used in the joins:
  • Match_tbl.league_id → League.id (foreign key to primary key)
  • League.country_id → Country.id (foreign key to primary key)

Full SQL (copy into DBeaver):

SELECT c.name AS country,
  AVG((m.home_team_goal + m.away_team_goal)::numeric) AS avg_goals_per_match
FROM soccer_proj."Match_tbl" m
JOIN soccer_proj."League" l ON l.id = m.league_id
JOIN soccer_proj."Country" c ON c.id = l.country_id
GROUP BY c.name
ORDER BY avg_goals_per_match DESC;

What to say on camera (example):
  "I join matches to leagues to countries, add home and away goals per match, take the
   average per country, and sort highest first. No CASE — just join, group, and AVG."

-------------------------------------------------------------------------------
Q18 — ADVANCED (JOIN with OR + conditional SUM)
-------------------------------------------------------------------------------

Plain English question:
  Which teams most frequently kept clean sheets?

What the answer looks like:
  One row per team_long_name with clean_sheets = total matches where that team conceded 0
  (whether they played home or away). Higher number = more clean sheets in the dataset.

How this query works (no WITH / no window functions):

  JOIN ... ON home_team_api_id = t.team_api_id OR away_team_api_id = t.team_api_id
    - Each team row links to every match they played, home or away.
    - OR in the join is why one team can match many Match_tbl rows.

  First CASE (inside SUM):
    - If this team was home AND away_team_goal = 0, add 1 (clean sheet at home).

  Second CASE (inside SUM):
    - If this team was away AND home_team_goal = 0, add 1 (clean sheet away).

  SUM(...) adds both contributions per match row for that team.
  GROUP BY team_long_name gives one total per team.
  ORDER BY clean_sheets DESC shows teams with the most clean sheets first.

Full SQL (copy into DBeaver):

SELECT t.team_long_name,
  SUM(CASE WHEN m.home_team_api_id = t.team_api_id AND m.away_team_goal = 0 THEN 1 ELSE 0 END
     + CASE WHEN m.away_team_api_id = t.team_api_id AND m.home_team_goal = 0 THEN 1 ELSE 0 END) AS clean_sheets
FROM soccer_proj."Team" t
JOIN soccer_proj."Match_tbl" m
  ON m.home_team_api_id = t.team_api_id OR m.away_team_api_id = t.team_api_id
GROUP BY t.team_long_name
ORDER BY clean_sheets DESC
LIMIT 30;

What to say on camera (example):
  "I join Team to Match_tbl where the team is either home or away. Each CASE adds 1 when
   the opponent did not score. I sum those per team name and sort to see who kept the most clean sheets."
