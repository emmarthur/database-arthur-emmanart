Multiple Choice

1. Which of the following is considered a DDL statement in your slides?
A) SELECT
B) INSERT
C) CREATE TABLE
D) DELETE

C)
Feedback: Correct. CREATE TABLE is DDL because it defines database objects.
Grade: Correct

2. Which clause is used to filter rows based on a condition?
A) ORDER BY
B) WHERE
C) LIMIT
D) OFFSET

B)
Feedback: Correct. WHERE filters rows based on conditions.
Grade: Correct

3. What does DISTINCT do?
A) Sorts rows in descending order
B) Removes duplicate rows from the result set
C) Filters NULL values only
D) Renames a column

B)
Feedback: Correct. DISTINCT removes duplicate rows in the selected result.
Grade: Correct

4. In PostgreSQL, which operator is case-insensitive for pattern matching?
A) LIKE
B) ILIKE
C) IN
D) BETWEEN

ILIKE
Feedback: Correct. In PostgreSQL, ILIKE is case-insensitive; LIKE is case-sensitive.
Grade: Correct

5. Which is the correct way to check for missing hire dates?
A) WHERE hire_date = NULL
B) WHERE hire_date <> NULL
C) WHERE hire_date IS NULL
D) WHERE hire_date LIKE NULL

C)
Feedback: Correct. NULL comparisons use IS NULL / IS NOT NULL, not = or <>.
Grade: Correct
