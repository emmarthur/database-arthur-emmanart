1) Write down the names of the 5 ANSI joins and what each does.
- INNER JOIN: only keeps rows where both tables match.
- LEFT (OUTER) JOIN: keeps all rows from the left table + matching rows from the right table (no match = NULL on the right).
- RIGHT (OUTER) JOIN: keeps all rows from the right table + matching rows from the left table (no match = NULL on the left).
- FULL (OUTER) JOIN: keeps all rows from both tables; matches where possible, NULLs where there is no match.
- CROSS JOIN: every row in table A paired with every row in table B (cartesian product).

2) What are the rules for using UNION, INTERSECT, and EXCEPT?
- Same number of columns in each SELECT result set.
- Corresponding columns must have compatible data types.
- Final output column names come from the first SELECT.
- UNION removes duplicate rows; UNION ALL keeps duplicate rows.
- UNION can have a performance hit vs UNION ALL because duplicate removal takes extra work.
- INTERSECT returns only rows common to both result sets.
- EXCEPT removes from the left result set any rows that also appear in the right result set.
- ORDER BY goes at the end of the full set operation query.

3) SQL scripts using students + club_members

-- List names and clubs for students who are in a club.
select s.name, c.club_name
from students as s
inner join club_members as c
on s.student_id = c.student_id;

-- List all students, whether or not they are in a club.
select s.student_id, s.name, s.major, c.club_name
from students as s
left join club_members as c
on s.student_id = c.student_id;

-- List students who are not in any club.
select s.student_id, s.name, s.major
from students as s
left join club_members as c
on s.student_id = c.student_id
where c.student_id is null;

-- List all students and club members (even if not matched).
select s.student_id as student_id
, s.name
, s.major
, c.student_id as club_student_id
, c.club_name
from students as s
full outer join club_members as c
on s.student_id = c.student_id;

-- Show every possible combination of students and clubs.
select s.name, s.major, c.club_name
from students as s
cross join club_members as c;

5a) Find the team name for all teams with at least one agent who speaks Cherokee.
-- full query (this is what the row count should match)
select distinct t.name as team_name
from team as t
inner join teamrel as tr
on t.team_id = tr.team_id
inner join agent as a
on tr.agent_id = a.agent_id
inner join languagerel as lr
on a.agent_id = lr.agent_id
inner join language as lang
on lr.lang_id = lang.lang_id
where lower(lang.language) = 'cherokee';
